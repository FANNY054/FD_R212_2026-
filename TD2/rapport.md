# Rapport — TD2

## Points clés

### 1. Apprentissage grâce au jeu 

Avec le jeu Flexbox Froggy j'ai pu utiliser différentes propriétés. Par exemple :  justify-content, align-items ou meme flex-wrap. Cela m'a permit de créer une page simple en css.

### 2. CSS et Html

J'ai appris également que le CSS sert à améliorer un html déjà propre, pas à corriger une mauvaise structure, il faut donc d'abord analyser le html avant de passer au CSS. De plus, faire un CSS simple et lisible est important pour que la page reste bien même si le contenu change.
