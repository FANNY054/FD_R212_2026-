## Points clés

### 1. Les différences entre HTML, CSS et Javascript

On a vu que :
- Html constitue toute la structure d'une page web. Il inclut les textes, titres et les différentes parties du site.
- Css lui, permet de styliser une page web. Il a un rôle esthétique.
- Et Javascript permet de concevoir un site interactif. Le site devient dynamique, contrôle le contenu multimédia...

### 2. Les différentes sections d'un site 

On a vu également les différentes balises qui constituent un site internet. Par exemple <main> c'est le contenu principal, <header> pour la navigation principal du site, <nav> pour le menu...
