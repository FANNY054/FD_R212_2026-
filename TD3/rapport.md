# Rapport — TD3

## Points clés

### 1. Explication modéle événement, état et rendu
Ce modele décrit la mécanique des intéraction d'une page web. 
Par Exemple avec Netflix. Il est posible d'enregistrer dans une liste des series et films. 
Donc événement -> l'utilisateur clique sur le bouton "ma liste" d'un film/série 
    Etat -> l'élément estDansLaListe passe de false à true
    Rendu -> Le bouton change, le bouton passe de + à ✔️ et le film/série apparait daans "Ma liste"
    
### 2. [classList.toggle est la passerelle entre JS et CSS]
classList.toggle est le lien entre JS et CSS. 
Chaque language à son rôle :
JS décide quand quelque chose change et CSS définit comment ça rend. 
Quand un événement se produit, le JS ajoute ou retire une classe sur un élément. C'est après le CSS qui s'occupe de changer l'apparence. 

"Aucune IA utilisée lors de cette séance."
